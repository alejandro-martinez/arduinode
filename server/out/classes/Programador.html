Module: Programador de Tareas

Clase (Singleton) para ABM de tareas. El modulo schedule permite programar las tareas; Permite: 1) Crear, eliminar, modificar y reprogramar tareas; 2) Observar ejecucion de tareas: Se relanzan cada x tiempo, (definido en tiempoEscaneoTareas) si se corta el servicio;
